# Scoutpost skills

Scoutpost publishes two public skill files for AI agents.

## 1. Product skill

URL: https://www.scoutpost.ai/skills/cojournalist.md

Use this when the agent should operate the product: connect through MCP or CLI, create scouts, search units, export findings, and respect editorial verification.

## 2. Setup skill

URL: https://www.scoutpost.ai/skills/cojournalist-setup.md

Use this when the agent should deploy or self-host Scoutpost on the user's own infrastructure.
